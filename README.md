# ChatGPT Saver Chrome Extension

This Chrome extension allows users to save their ChatGPT chats into a text file on their local system. It extracts the questions and answers from the chats and saves them in an easy-to-read format.

## Features

- Extracts questions and answers from ChatGPT chats.  Will work on one current chat you select at a time. 
- Saves the extracted content as a `.txt` file on your local system
- Adds an extra line between the question and answer for better readability

## How to use

1. Clone this repository or download the source code as a ZIP file.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" by toggling the switch in the top right corner.
4. Click "Load unpacked" and select the folder containing the source code of this extension.
5. The ChatGPT Saver extension should now appear in your list of extensions.
6. To use the extension, navigate to the ChatGPT website where your chats are displayed.
7. Click the extension icon in your browser's toolbar, and then click "Save ChatGPT Chats" button.
8. A `.txt` file containing the extracted questions and answers will be downloaded to your local system.

## Notes

- This extension is specifically designed for use with the ChatGPT platform. It may not work with other platforms or future updates
